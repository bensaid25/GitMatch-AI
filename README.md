

GitHub Analyzer PRO

A high-performance technical evaluation dashboard built with React. This tool helps HR managers and developers analyze GitHub profiles through data-driven insights.

Features
* **Dual-User Personas:** Distinct workspaces for 'Visitors' and 'Recruiters'.
* **Skill Radar:** Automated language distribution analysis.
* **Internship Matcher:** Keyword-based compatibility engine for job descriptions.
* **Pro Report Export:** One-click PDF generation for candidate evaluation.
* **Glassmorphism UI:** Modern, responsive design with shimmering "Sparkle" effects.

Tech Stack
* **Frontend:** React.js (Vite)
* **Styling:** CSS3 (Custom Glassmorphism & Animations)
* **API:** GitHub REST API

Getting Started
1. Clone the repo: `git clone [your-repo-link]`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
